Feature barcode reference generator

Builds the antibody barcode CSV required for CITE-seq, DOGMA-seq, ASAP-seq, and Tapestri experiments.

Search the BioLegend barcode library by marker name, clone, or TotalSeq ID, then export a reference file in your required format. Antibody lists are sourced directly from the BioLegend barcode lookup and reformatted for pipeline use. Where possible, markers have been standardised to their CD numbers — see the human and mouse CD lists for reference. You can verify a clone or catalogue number on the BioLegend website.

Supported output formats:

  • Cell Ranger — for CITE-seq, DOGMA-seq, and other 10x RNA + antibody assays
  • Kallisto KITE — for ASAP-seq
  • Tapestri — for DNA and protein sequencing
