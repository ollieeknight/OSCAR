# Interactive Tools

OSCAR provides two interactive web tools to help you generate properly formatted files for your single-cell analysis workflows.

## Available Tools

### [Metadata Generator](metadata-generator.html)
Create standardized metadata CSV files for your experiments. This tool guides you through all the required fields and ensures your metadata is correctly formatted for OSCAR processing.

**Use this tool to:**

- Generate metadata files for single-cell experiments
- Ensure consistent formatting across projects
- Validate required fields before processing

### [Feature Barcode Generator](adt-generator.html)
Generate feature barcode reference files for Cell Ranger, Kallisto (KITE), and Tapestri workflows. Select from TotalSeq antibody panels and export in the correct format for your platform.

**Use this tool to:**

- Create ADT/HTO reference files
- Search BioLegend TotalSeq panels
- Export in Cell Ranger, Kallisto, or Tapestri format
