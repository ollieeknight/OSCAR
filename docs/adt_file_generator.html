OSCAR ADT List Generator

This page allows you to generate an ADT list file for use with the Romagnani lab OSCAR pipeline.
This pipeline takes raw sequencing data and performs cellranger counting, as well as quality control, doublet detection, and donor genotype deconvolution, as detailed here.

Several input variables are required to make the ADT list file. These are detailed below:

  • ADT Name: The name of the ADT
  • ADT Sequence: The sequence of the ADT
  • ADT Barcode: The barcode of the ADT
