OSCAR Metadata Generator

Introduction

This page allows you to generate metadata for the OSCAR project. Fill in the required fields and generate a CSV file for your records.

Background

The OSCAR project aims to streamline the process of metadata generation for various assays. This tool helps in organizing and exporting the necessary information in a structured format.
