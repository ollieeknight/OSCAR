OSCAR ADT List Generator

This page allows you to generate an ADT list file for use with the Romagnani lab OSCAR pipeline for sequencing data analysis.
This pipeline takes raw sequencing data and performs cellranger counting, as well as quality control, doublet detection, and donor genotype deconvolution, as detailed in the OSCAR pipeline documentation.
