Feature barcode reference generator

This page allows you to generate a feature barcode file for Cell Ranger and (Kallisto Indexing and Tag Extractor (KITE)).
Antibody lists are taken directly from the BioLegend barcode lookup website and slightly reformatted in order to search for the correct names. In a few cases, markers will have been renamed to their CD numbers (mouse and human), but in any case you can search for the antibody clone and catalogue number on the BioLegend website.
Format choices:

  • For processing a 10x scRNA-seq experiment (3prime, 5prime, DOGMA), please select the 'Cellranger' output format.
  • For an ASAP-seq experiment, you'll process your ADT FASTQ files with Kallisto.
  • For uploading tapestri FASTQ files, please select the 'Tapestri' output format.
